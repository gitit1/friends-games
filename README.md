# 🌟 עולם החברים — משחקי חשיבה בעברית

אתר משחקים חינוכי בעברית — RTL מלא, מובייל־פירסט — לילדים צעירים:
**52 משחקים** ב־8 קטגוריות (מספרים, חשיבה, יצירה, אותיות, אנגלית, ספורט, כיף, וגדילה),
**100 "חברים"** צבעוניים עם קול עברי מוקלט, ומשחק תלת־ממד אחד (three.js).

🎮 **המשחקים באוויר:** https://assaf-friends-games.netlify.app

## עקרונות העיצוב

- **בלי כישלונות** — אין game over, אין טיימרים, אין תחרות; טעות מקבלת רמז עדין וממשיכים.
- **קושי מצטבר** — כל משחק נפתח קל ומסתיים ברמת "אלוף" שכוללת הכול.
- **רגוע לחושים** — צבעים מעומעמים, בלי הבזקים, אנימציה קצרה וקשורה לפעולת הילד, כיבוד reduced-motion.
- **טרום־קוראים** — הכול תמונות + הקראה בעברית (קול מוקלט + נפילה ל־TTS).
- **עברית ואנגלית** — ממשק דו־לשוני עם שיקוף RTL/LTR מלא.
- **בקרת הורים** — רמות, טווח מספרים, פעולות חשבון, קול, מצב תנועה רגועה ועוד.

## טכנולוגיה

React 19 · TypeScript · Vite · three.js (משחק התלת־ממד בלבד, בטעינה עצלה) · ללא ספריות UI.
