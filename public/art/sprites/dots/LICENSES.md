## Provenance / processing

Every sprite is procedurally baked by `scripts/gen-dots-art.mjs` — an original
`@napi-rs/canvas` bake with seeded value-noise material grain and analytic baked
lighting/AO (the same pipeline as the dice, sortshelf and coin-sort materials).
All colours are muted, desaturated tones per the app's sensory-accessibility rule
(no neon, no fully-saturated red). The connecting stroke is drawn at runtime as an
SVG crayon path (roughened + soft wax bloom) and the numbers stay crisp CSS text —
those are render materials layered over the baked surfaces, never flat CSS panels.
The build tool's only dependency (`@napi-rs/canvas`) is already installed and is
**not** shipped with the app.
