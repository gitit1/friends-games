## Provenance / processing

Every sprite is procedurally baked with a transparent background by
`scripts/gen-letterhunt-art.mjs` — an original `@napi-rs/canvas` bake with seeded
value-noise material grain and analytic baked lighting/AO (the same pipeline as
the dice / who-is-missing / coin-sort materials). Colours are muted pastels per
the app's sensory-accessibility rule. The build tool's only dependency
(`@napi-rs/canvas`) is already pinned in the app's package.json.

The block's FRONT-FACE panel occupies a fixed sub-rectangle of the sprite
(`FX0`/`FY0`/`FS` in the bake script, mirrored by `.lh-block-ch` in `src/app.css`),
so the runtime CSS letter overlay lands centred on the baked panel.
