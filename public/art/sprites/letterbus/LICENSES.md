## Provenance / processing

Every sprite is procedurally baked with a transparent background by
`scripts/gen-letterbus-art.mjs` — an original `@napi-rs/canvas` bake with seeded
value-noise material grain, analytic shape SDFs and baked cylindrical/dome
lighting, AO and contact shadow (the same pipeline as the car-race, coin-sort and
dice materials). Colours are muted per the app's sensory-accessibility rule, with
one warm hero colour (the bus) against a warm-grey road and sage town backdrop.
The build tool's only dependency (`@napi-rs/canvas`) is already pinned in
`package.json`.

The bus's passenger WINDOW BAND is a fixed sub-rectangle of the sprite (`WIN` in
the bake script → `--lb-win-*` in `src/app.css`), centred on the bus so the crisp
Hebrew letter riders overlay pixel-aligned in both Hebrew and the mirrored English
layout. Wheel arches are symmetric about the bus centre so one wheel position
serves both. The Hebrew LETTERS themselves stay crisp CSS text (the shared
`LetterGuy`) on top — never baked — so they read razor-sharp.
