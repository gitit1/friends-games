## Provenance / processing

Every sprite is procedurally baked with a transparent background by
`scripts/gen-sortshelf-art.mjs` — an original `@napi-rs/canvas` bake with seeded
value-noise material grain and analytic baked lighting/AO (the same pipeline as
the dice, pattern and coin-sort materials). All colours are muted, desaturated
tones per the app's sensory-accessibility rule (no neon, no fully-saturated red).
The build tool's only dependency (`@napi-rs/canvas`) is already installed and is
**not** shipped with the app.
